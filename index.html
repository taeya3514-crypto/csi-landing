USP ①

엑셀 입력 → 봇이 전부 처리

기존 엑셀 양식에 데이터를 채우면 봇이 CSI에 직접 입력합니다. 종목 검색, 항목 매핑, PDF 첨부, 제출까지 사람 손 필요 없습니다.

  • 시험종목 코드 자동 검색 · 매핑

  • 등록 성공/실패 실시간 결과 리포트
  • 네트워크 오류 · 팝업 자동 재시도

USP ②

어떤 현장이든

장기 투입 현장은 전용봇(현장 정보 내장, 빠른 실행), 단기·이동 현장은 범용봇(ok.xlsx 하나로 현장 전환)으로 대응합니다.
